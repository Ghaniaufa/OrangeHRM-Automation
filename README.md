Repository ini berisi kumpulan project Quality Assurance (QA) Automation yang saya kerjakan selama mengikuti Bootcamp QA Automation SanberCode. Project ini mencakup pengujian manual maupun otomatis pada aplikasi web OrangeHRM Demo serta pengujian API menggunakan Postman dan Cypress.

Seluruh automation dibangun menggunakan konsep Page Object Model (POM) untuk meningkatkan keterbacaan dan kemudahan pemeliharaan kode. Selain itu, project ini juga menerapkan Cypress Intercept untuk memvalidasi network request serta menghasilkan laporan pengujian menggunakan Newman HTML Report.
